
> **Purpose**: A clean, mentor‑grade, future‑public reference of _standard, industry‑accepted tools_ used across ethical hacking and cybersecurity domains.  
> **Design goal**: Crisp, minimal, no fluff, no random tools.

---

## 1. Network Reconnaissance & Packet Analysis

Tools used to observe, analyze, and map network traffic and behavior.

- **Wireshark** – Deep packet inspection and protocol analysis
    
- **tcpdump** – CLI packet capture and analysis
    
- **Netcat (nc)** – Network debugging, banner grabbing, pivoting
    
- **Scapy** – Packet crafting and network testing (Python‑based)
    

---

## 2. Network & Port Scanning (Reconnaissance)

Tools to identify live hosts, open ports, services, and OS fingerprints.

- **Nmap** – Industry standard network and service scanner
    
- **Masscan** – High‑speed internet‑scale port scanning
    
- **Angry IP Scanner** – Lightweight IP/port discovery
    

---

## 3. Web Application Security Testing

Tools for identifying vulnerabilities in web apps and APIs.

- **Burp Suite** – Interception, fuzzing, scanning, manual exploitation
    
- **OWASP ZAP** – Open‑source web vulnerability scanner
    
- **Nikto** – Web server misconfiguration and vulnerability scanner
    
- **SQLmap** – Automated SQL injection exploitation
    

---

## 4. Vulnerability Assessment & Scanning

Tools used to discover known vulnerabilities in systems and applications.

- **Nessus** – Enterprise vulnerability scanner
    
- **OpenVAS / Greenbone** – Open‑source vulnerability assessment
    
- **Qualys** – Cloud‑based enterprise vulnerability management
    

---

## 5. Exploitation Frameworks

Tools used to develop, test, and execute exploits in a controlled manner.

- **Metasploit Framework** – Exploit development and post‑exploitation
    
- **BeEF** – Browser exploitation framework
    
- **Searchsploit** – Offline exploit database (Exploit‑DB)
    

---

## 6. Password Attacks & Credential Testing

Tools for auditing password strength and authentication mechanisms.

- **John the Ripper** – Password cracking (CPU/GPU)
    
- **Hashcat** – High‑performance password cracking
    
- **Hydra** – Online login brute‑force testing
    

---

## 7. Wireless & Network Attacks

Tools focused on Wi‑Fi, Bluetooth, and RF security testing.

- **Aircrack‑ng** – Wi‑Fi auditing and attacks
    
- **Kismet** – Wireless discovery and monitoring
    
- **Reaver** – WPS attack testing
    

---

## 8. Reverse Engineering & Binary Analysis

Tools used to analyze compiled binaries, malware, and firmware.

- **Ghidra** – NSA‑released reverse engineering suite
    
- **IDA Free / Pro** – Disassembler and debugger
    
- **Radare2** – CLI‑based reverse engineering framework
    

---

## 9. Malware Analysis & Forensics

Tools for static and dynamic analysis of malicious code.

- **x64dbg / OllyDbg** – Runtime debugging
    
- **Process Monitor** – System activity tracking
    
- **Volatility** – Memory forensics framework
    

---

## 10. Operating Systems & Environments

Specialized OS platforms for penetration testing and labs.

- **Kali Linux** – Primary penetration testing OS
    
- **Parrot Security OS** – Lightweight security‑focused OS
    
- **Metasploitable** – Intentionally vulnerable testing VM
    

---

## 12. Cloud & Modern Attack Surface (Essential)

Modern security testing across cloud infrastructure.

- **ScoutSuite** – Cloud security auditing
    
- **Prowler** – AWS security best practices assessment
    
- **CloudSploit** – Cloud misconfiguration scanning
